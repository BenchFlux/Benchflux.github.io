Tutorials

The Benchflux project is a consortium of individual research teams working on their own pipeline. Below, you can explore each team's individual contribution. The Benchflux project aims to integrate these working groups' data into a scale-aware data pipeline. These tutorials will be hosted on our Github Organization.

ALIVE

Paul Stoy · Sadegh Ranjbar

Low-latency, multi-scale flux estimation using geostationary remote sensing; builds gap-free EO layers and compares multiple ML methods for continental-scale, hourly datasets with uncertainties and sensitivities.
